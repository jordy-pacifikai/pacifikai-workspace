📣 CONTEXTE

L'Annonce qui change tout

Le 17 fevrier 2026, Figma et Anthropic annoncent "Code to Canvas" — la passerelle entre le code IA et le design collaboratif.

Qu'est-ce que Code to Canvas ?

Une integration native entre Claude Code (l'outil de coding IA d'Anthropic) et Figma (la plateforme de design collaborative). Le code genere par l'IA se transforme automatiquement en layers Figma editables.

💡 Le detail technique

Le Figma MCP Server fonctionne comme un pont entre l'environnement de code et le canvas. Il prend le DOM rendu du navigateur (pas le code source) et le traduit en layers Figma natifs : textes, formes, groupes, composants. Chaque element est editable independamment — c'est pas un screenshot, c'est un vrai fichier Figma.

⚠ RISQUES

Risques & Mitigations

Chaque risque identifie avec sa strategie d'attenuation. Approche pragmatique.

🎨

Qualite design pas assez "pro"

Claude Code genere du code fonctionnel mais le design pourrait manquer de polish pour certains clients exigeants.

Mitigation : Utiliser shadcn/ui + Tailwind presets + skill /frontend-design pour un rendu premium systematique
🔄

Client veut des modifs complexes post-Figma

Le client pourrait vouloir modifier des elements dans Figma qui ne se retranscrivent pas facilement dans le code.

Mitigation : Le Figma est un point de validation, pas le deliverable final. Les ajustements se font dans le code, avec re-export si necessaire
👥

Concurrence qui copie l'approche

D'autres agences polynesienne pourraient adopter les memes outils IA a terme.

Mitigation : Notre avance c'est la stack complete (Claude Code + n8n + Supabase + Vercel + MCP), pas juste le design. L'expertise est notre moat
🔒

Dependance a Anthropic/Claude

Si Anthropic change ses tarifs ou restreint l'acces, notre modele est impacte.

Mitigation : Diversifier vers Cursor/Codex si necessaire. Le MCP est un standard ouvert, pas proprietaire
🌐

Connectivite internet en PF

La latence reseau en Polynesie peut affecter l'experience "live" pendant les demos.

Mitigation : Pre-generer des templates avant les RDV. Utiliser le mode desktop Figma MCP pour un workflow local
