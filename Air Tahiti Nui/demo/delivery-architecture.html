Questions Frequentes

Reponses directes a tes interrogations

Comment recuperer les donnees client ?

Google Sheets partage (Phase 1) ou API directe (Phase 2 - Amadeus/Sabre)

Ou stocker les outputs ?

Airtable pour business data (newsletters, articles, KPIs) + Supabase pour technique (vectors, memory)

Airtable est-il securise ?

OUI - SOC 2, AES-256, RGPD conforme. Upgrade Team ($20/mois) pour audit logs complets.

Grafana est-il necessaire ?

NON pour l'instant. n8n logs + Airtable Dashboard suffisent pour 1-5 clients.

Faut-il un dashboard + calendrier ?

OUI mais utilise Airtable Calendar View (gratuit) - pas besoin de le coder.

Comment isoler les clients ?

1 base Airtable DEDIEE par client + prefixe [CLIENT] sur chaque workflow n8n.
